What is this?

This portal lets DMV staff register people, register their vehicles, transfer ownership, and issue driver licenses. Every action goes through business rules (age limits, unique VINs) and is stored in a real SQL database running inside the app.

Curious what the database looks like? Open the DB console (JDBC URL jdbc:h2:mem:dmv, user sa, no password) to browse the tables Hibernate created from our Java classes.
